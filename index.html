HTML

  • HTML is a way of building a framework for our webpage
  • It uses elements to structure a page
  • head describes the characterists of the page and are unseen by users
  • body describes the content of the page

CSS

  • CSS helps add style to our page
  • CSS sytles can be added internally, externally, or inline
  • internally are general sytles applied to all elements of a kind specified written within the main code
  • externally are general sytles applied ot all elements specified but is stored externally in a seperate file and referenced by main
  • inline are styles added directly to a specific element in main

GIT

  • GIT is a way of storing projects in the cloud allowing teams to work in tandom or seperately on a single project
  • The GIT work flow follows the path commit, push, pull
  • commit captures the changes made to a project to be refered to later
  • push adds these changes to our working branch
  • pull requests allow the team to inspect the new feature before it is merged
  • once approved, it is merged into the main

JavaScript

  • language used to add functionality to our structure
  • Variables are containers that hold data
  • Control flow executes code in order top to bottom
